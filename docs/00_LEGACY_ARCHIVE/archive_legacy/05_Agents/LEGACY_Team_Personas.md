# LEGACY Team Personas

- Performance optimization

## Team Collaboration

### Communication Channels
- **Technical discussions**: GitHub issues and pull requests
- **General communication**: Discord/Slack community
- **Documentation**: GitHub wiki and project docs
- **User feedback**: GitHub issues and community forums

### Decision Making Process
1. **Technical decisions**: Lead Architect with team input
2. **Feature priorities**: Product Manager with user feedback
3. **Design choices**: UI/UX Designer with user testing
4. **Security decisions**: Security Specialist with team review

### Development Workflow
1. **Planning**: Product Manager creates requirements
2. **Design**: UI/UX Designer creates mockups
3. **Architecture**: Lead Architect defines technical approach
4. **Development**: Frontend and Backend Developers implement
5. **Testing**: QA Engineer validates functionality
6. **Documentation**: Technical Writer creates guides
7. **Release**: DevOps Engineer manages deployment
