**Title: HIGH-TICKET ROADMAP**

* **Phase 1: Foundation (Current):** Migrate legacy `g4ScoreBoard` files into the new `/src` modular architecture.
* **Phase 1 Goal:** Establish the **Dexie.js** database schema and the **BroadcastChannel** messaging bus as the core engine.
* **Phase 2: The Billiards Suite:** Complete the transition of the Billiards master dock and scoreboard overlay to the new data flow.
* **Phase 3: Sponsor Engine:** Build the integrated ad manager to track and display sponsor logos with data-driven impression counts.
* **Phase 4: Expansion:** Develop the "Darts" feature module by utilizing the existing `/core` engine without rewriting the base code.
* **Phase 5: Automation:** Implement **OBS-WebSocket** integration to allow the scoreboard to trigger scene changes or camera switches automatically.
