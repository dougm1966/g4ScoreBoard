# Media Directory Structure

This folder contains all images for the G4ScoreBoard system.

## Directory Structure:
- `players/` - Player photos
- `sponsors/` - Sponsor logos
- `slideshow/` - Slideshow logos
- `ads/` - Future advertising images

## File Naming Convention:
- Player photos: `player1_photo.png`, `player2_photo.png`
- Sponsor logos: `left_sponsor_logo.png`, `right_sponsor_logo.png`
- Slideshow logos: `custom_logo_1.png`, `custom_logo_2.png`, `custom_logo_3.png`

## Access Method:
Images are accessed via relative paths from the project root:
`./media/images/players/player1_photo.png`
