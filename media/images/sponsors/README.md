# Sponsor Logos Directory

Sponsor logos are stored here with the following naming convention:
- `left_sponsor_logo.png` - Left sponsor logo
- `right_sponsor_logo.png` - Right sponsor logo
- `custom_logo.png` - Browser compact custom logo

## Access Paths:
- `./media/images/sponsors/left_sponsor_logo.png`
- `./media/images/sponsors/right_sponsor_logo.png`
- `./media/images/sponsors/custom_logo.png`
