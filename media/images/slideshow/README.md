# Slideshow Logos Directory

Slideshow logos are stored here with the following naming convention:
- `custom_logo_1.png` - Slideshow logo 1
- `custom_logo_2.png` - Slideshow logo 2
- `custom_logo_3.png` - Slideshow logo 3

## Access Paths:
- `./media/images/slideshow/custom_logo_1.png`
- `./media/images/slideshow/custom_logo_2.png`
- `./media/images/slideshow/custom_logo_3.png`
